## Базовый адрес

В примерах используется:

```text
http://<host>:51351
```

Замените <host> реальным IP или доменом. В production рекомендуется HTTPS через reverse proxy.

## OpenAPI

Черновая схема основных маршрутов доступна по адресу:

```text
http://<host>:51351/api/openapi.json
```

Файлы cookie, public_token и NODE_AGENT_TOKEN дают доступ к данным и операциям. Не вставляйте их в публичные тикеты и команды, которые сохраняются в общей истории shell.
