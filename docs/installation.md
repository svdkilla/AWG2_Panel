# Установка

До установки выберите архитектуру. Для одного региона достаточно локальной схемы; для нескольких стран используйте главную панель и ноды.

## Требования

- Ubuntu 22.04/24.04 или Debian 11–13;
- root-доступ;
- запущенный Docker;
- заранее установленный контейнер AmneziaWG 2.0;
- открытый TCP-порт панели, по умолчанию 51351;
- открытый UDP-порт AmneziaWG;
- сетевой доступ главной панели к agent API нод.

Скрипт install.sh при необходимости установит docker.io и openssl, но не устанавливает AmneziaWG.

## Проверка AmneziaWG

```bash
docker ps --filter name=amnezia-awg2
```

Ожидается запущенный контейнер с проброшенным UDP-портом:

```text
amnezia-awg2   Up ...   0.0.0.0:<udp-port>-><udp-port>/udp
```

## Рабочий каталог

Рекомендуемый путь:

```text
/opt/xconnect-awg-panel
```

Критичные данные находятся в data:

ПутьНазначениеdata/panel.envсекреты и параметры запускаdata/panel.sqlite3пользователи, устройства, админы, ноды и брендингdata/backupsархивы резервных копий

Не отправляйте panel.env клиентам и не добавляйте его в публичный репозиторий. Файл содержит пароль первого владельца, секрет сессий и токен agent API.
