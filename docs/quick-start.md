# Быстрый старт

Этот сценарий подходит для уже установленной панели. За несколько минут вы войдёте в админку, создадите клиента и проверите конфигурацию.

## 1. Проверьте сервисы

```bash
docker ps --filter name=xconnect-awg-panel
docker ps --filter name=amnezia-awg2
```

Оба контейнера должны иметь состояние Up. Если панель не запущена, откройте раздел «Диагностика».

## 2. Получите пароль первого владельца

```bash
grep '^ADMIN_PASSWORD=' /opt/xconnect-awg-panel/data/panel.env
```

Откройте http://<server-ip>:51351 и войдите под пользователем admin.

Пароль из panel.env предназначен для первого входа. После проверки создайте собственный надёжный пароль владельца и ограничьте доступ к панели.

## 3. Создайте пользователя

1. Откройте раздел «Пользователи».
2. Нажмите кнопку добавления.
3. Укажите уникальный username.
4. Задайте дату окончания подписки.
5. Укажите лимит устройств и трафика. Значение 0 отключает соответствующий лимит.
6. Для мультисерверной схемы выберите доступные страны.
7. Сохраните пользователя.

## 4. Добавьте устройство

Откройте карточку пользователя, создайте устройство и дайте ему понятное имя, например iPhone или Home Router.

Если страна не выбрана, устройство создаётся на всех разрешённых нодах пользователя. Если выбрана конкретная нода — конфигурация будет выпущена только для неё.

## 5. Передайте доступ клиенту

Используйте один из вариантов:

- ссылку вида http://<host>:51351/u/<public_token>;
- QR с vpn:// — основной способ для AmneziaWG 2.0;
- файл .conf;
- QR с raw-конфигурацией как запасной вариант.

## 6. Проверьте подключение

```bash
docker exec -it amnezia-awg2 sh -lc 'awg show awg0'
```

После подключения клиента у peer должны обновиться handshake и счётчики трафика.
